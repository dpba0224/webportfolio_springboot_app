InvoEase (Invoice Generator App)

A full-stack web application that generates invoice that fits the needs of most business using Spring Boot, React and MongoDB. This can help improve the efficiency of current workflows of businesses that generates invoice. In addition, users can save, download a .pdf copy of invoice, and send a copy to an e-mail address for their own reference. Authentication is supported using Clerk to ensure that all users can log in with ease and secure while using this application.

Skills Applied:
Spring Boot MongoDB Lombok Axios React CSS Bootstrap
